Function benchmark

簡易ベンチマークを取る

「指定ミリ秒内で何回コールできるか？」でベンチする。

$suite は ['表示名' => $callable] 形式の配列。 表示名が与えられていない場合、それらしい名前で表示する。

Example:

// intval と int キャストはどちらが早いか調べる
benchmark([
    'intval',
    'intcast' => function($v){return (int)$v;},
], ['12345'], 10);
