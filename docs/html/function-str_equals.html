Function str_equals

文字列比較の関数版

文字列以外が与えられた場合は常に false を返す。ただし __toString を実装したオブジェクトは別。

Example:

assert(str_equals('abc', 'abc')       === true);
assert(str_equals('abc', 'ABC', true) === true);
assert(str_equals('\0abc', '\0abc')   === true);
