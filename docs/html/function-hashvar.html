Function hashvar

変数指定をできるようにした compact

名前空間指定の呼び出しは未対応。use して関数名だけで呼び出す必要がある。

Example:

$hoge = 'HOGE';
$fuga = 'FUGA';
assert(hashvar($hoge, $fuga) === ['hoge' => 'HOGE', 'fuga' => 'FUGA']);
