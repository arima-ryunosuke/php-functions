Function array_depth

配列の次元数を返す

フラット配列は 1 と定義する。 つまり、配列を与える限りは 0 以下を返すことはない。

Example:

assert(array_depth([])                       === 1);
assert(array_depth(['hoge'])                 === 1);
assert(array_depth([['nest1' => ['nest2']]]) === 3);
