Function try_catch_finally

try ～ catch ～ finally 構文の関数版

php < 5.5 にはないし、例外機構構文が冗長なことがまれによくあるはず。

Example:

$ex = new \Exception('try_catch');
assert(try_catch(function() use ($ex) { throw $ex; }) === $ex);
