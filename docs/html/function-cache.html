Function cache

シンプルにキャッシュする

この関数は get/set を兼ねる。 キャッシュがある場合はそれを返し、ない場合は $provider を呼び出してその結果をキャッシュしつつそれを返す。

内部キャッシュオブジェクトがあるならそれを使う。その場合リクエストを跨いでキャッシュされる。 内部キャッシュオブジェクトがないあるいは $use_internal=false なら素の static 変数でキャッシュする。

Example:

// 乱数を返す処理だが、キャッシュされるので同じ値になる
$rand1 = cache('rand', function(){return rand();});
$rand2 = cache('rand', function(){return rand();});
assert($rand1 === $rand2);
