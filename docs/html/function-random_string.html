Function random_string

安全な乱数文字列を生成する

下記のいずれかを記述順の優先度で使用する。

  • random_bytes: 汎用だが php7 以降のみ
  • openssl_random_pseudo_bytes: openSsl が必要
  • mcrypt_create_iv: Mcrypt が必要
